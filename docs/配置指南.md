# Sensor Tower MCP 配置指南

## 快速配置

在 MCP 配置文件中添加：

```json
{
  "mcpServers": {
    "sensortower": {
      "command": "npx",
      "args": ["-y", "sensor-tower-mcp-pro"],
      "env": {
        "SENSOR_TOWER_API_TOKEN": "你的主Token",
        "SENSOR_TOWER_API_TOKEN_BACKUP": "备用Token（可选）"
      }
    }
  }
}
```

## 环境变量

| 变量名 | 说明 |
|--------|------|
| `SENSOR_TOWER_API_TOKEN` | 主API Token（必填） |
| `SENSOR_TOWER_API_TOKEN_BACKUP` | 备用Token（可选） |
| `SENSOR_TOWER_API_TOKEN_2` ~ `_10` | 更多备用Token（可选） |

> Token额度用尽时会自动切换到备用Token
